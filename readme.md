# Personal Expense Tracker

## About the Project
Personal Expense Tracker is a simple mobile application developed to help users record, organize, and monitor their daily expenses in an easy and efficient way. This project also helps us practice and improve our mobile app development skills as beginner developers.

## Objectives
- To develop a basic mobile application.
- To allow users to track their daily expenses.
- To apply the concepts learned in class.
- To improve our programming and problem-solving skills.

## Features
- Add a new expense
- Edit an existing expense
- Delete an expense
- View all recorded expenses
- Calculate total spending
- Categorize expenses (Food, Transport, Bills, etc.)
