#  Login Screen Specifications

##  Branding & Colors
- **Primary Green:** `0xFF00C853` (Logo & Sign Up link)
- **Background:** `0xFFFFFFFF` (White)
- **Field Fill:** `0xFFE0E0E0` (Grey)
- **Main Button:** `0xFF000000` (Black)

##  Layout Details
- **Screen Size:** 393 x 852 (Designed for mobile)
- **Header:** Center the Logo (SVG) and "Welcome Back!" text.
- **Inputs:** Use `TextFormField` with rounded corners.
- **Action:** Black button should span the full width of the form.

##  Assets Needed
- `logo.svg`
- `ic_eye.svg` (Password visibility)
