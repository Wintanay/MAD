# UI/UX Design Documentation 🎨

## 📱 App Screens
1. **Home Screen:** Main dashboard with balance and recent transactions.
2. **Add Expense Screen:** Form to input new spending data.
3. **Transaction History:** Full list of all records.
4. **Statistics:** Charts showing spending habits.

## 🎨 Color Theme
* **Primary:** #2D2D2D (Dark Grey)
* **Action:** #4CAF50 (Green)
* **Alert:** #F44336 (Red)
