# UI/UX Design Documentation 

##  App Screens
1. **Entry Screen:** The very first thing the user sees (Logo + Branding).
2. **Login / Sign-up:** Where users enter their credentials to access their private data.
3. **Home Screen:** Main dashboard with balance and recent transactions.
4. **Add Expense Screen:** Form to input new spending data.
5. **History:** Full list of all records.
6. **Charts:** Charts showing spending habits.

##  Color Theme
**Primary (Deep Charcoal) - `#1A1C1E`**: This is our "Base" color. It is used for the **Balance Card**, headers, and main navigation to provide a modern, premium feel.
* **Success (Emerald Green) - `#00C853`**: This represents "Growth" and "Income." It is used for **deposit amounts**, "Add" buttons, and positive trends in the statistics.
* **Alert (Coral Red) - `#FF5252`**: This represents "Spending" and "Warnings." It is used for **expense amounts**, "Delete" actions, and when a category goes over budget.
* **Background (Pure White) - `#FFFFFF`**: Used for the main app background to ensure high contrast and readability for all financial data.
