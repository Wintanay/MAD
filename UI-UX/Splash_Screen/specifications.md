#  Splash Screen Specifications

##  Layout & Colors
- **Background:** `0xFF00C853` (Primary Success Green)
- **Center Logo Circle:** `0xFFFFFFFF` (Pure White)
- **App Name Text:** `0xFF000000` (Solid Black)

##  Layout Details
- **Main Background:** Full-screen solid green fill using the primary app theme color.
- **Icon Container:** A perfectly centered white circular shape (acting as a focal point).
- **App Icon:** A green wallet/credit card icon centered inside the white circle container.
- **Typography:** "Expense Tracker" in bold black Sans-Serif font, positioned directly below the center circle for clear branding.
- **Transition:** On launch, this screen should appear for 2-3 seconds before navigating to the Login/Home screen.
